# Security Policy

We take the security of `pgui` seriously. If you find any vulnerabilities or security concerns, please report them to us immediately.

## Supported Versions

Currently, the following versions receive security updates:

| Version | Supported |
| --- | --- |
| 0.1.x | :white_check_mark: Yes |
| < 0.1.0 | :x: No |

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities. Instead:
- Open a private GitHub security advisory for this repository if available.
- Otherwise, email the maintainer listed in the repository profile.

We will acknowledge your report within 48 hours and work with you to coordinate a security patch and disclosure.
