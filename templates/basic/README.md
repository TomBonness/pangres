## Quick start

To start the development container:
```sh
./run.sh
```

To run verification checks:
```sh
./verify.sh
```

## Structure
- `db/framework/pgui.sql`: The `pgui` SQL web framework source. To update the framework, simply copy a newer version of `pgui.sql` here.
- `db/app/`: Your application source:
  - `schema.sql`: Database schema tables and domains.
  - `handlers.sql`: HTTP request handler functions.
  - `routes.sql`: Application routes mapping paths to handler functions.
- `run.sh`: Start/stop/configure the Omnigres database container.
- `verify.sh`: Verify app routing, functionality, and security invariants.
